# rizzchat-master

Manual Next.js 14 + TypeScript + Tailwind setup (App Router, `src/` dir).

## Scripts
- dev: `npm run dev`
- build: `npm run build`
- start: `npm start`
- lint: `npm run lint`

## Getting started
1. Ensure Node.js LTS (>=20) is installed.
2. Install deps: `npm install`
3. Run dev: `npm run dev`
